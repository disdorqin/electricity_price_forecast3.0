# 实验运行约定

本项目的正式训练实验默认使用 GPU，并固定使用本机 conda 环境 `epf-2`。

```powershell
.\scripts\runners\run_train_epf2.ps1
```

默认入口是：

```powershell
python -m scripts.runners.run_badtpmixer_ra916_quick_multimonth --device cuda
```

环境位置：

```text
D:\computer_download\environment\conda\epf-2\python.exe
```

当前已确认：

```text
torch 2.5.1+cu121
CUDA available: True
GPU: NVIDIA GeForce RTX 4060 Laptop GPU
```

如果需要跑其他入口，把模块名作为第一个参数传入，后面继续接该 runner 的参数：

```powershell
.\scripts\runners\run_train_epf2.ps1 scripts.runners.run_badtpmixer_ra916_quick_multimonth --test_months 2024-04,2024-05,2024-06 --device cuda
```

RA916 当前实验方向：

- 停止后验 FOBO selector 路线。
- 不删除 FOBO 文件，FOBO 只作为 teacher / oracle 诊断数据。
- 主线切到 TriFactorSpikeTimesNet / BA-DTPMixer 主干内 9-16 Regime-Aware Head。
- 优先跑通 `RA916_Direct`、`RA916_Gated`、`RA916_RegimeGated`。
- 不允许 fallback week。
- `random_state=7`。

输出目录：

```text
outputs/BA_DTPMixer_RA916/quick_multimonth
```
