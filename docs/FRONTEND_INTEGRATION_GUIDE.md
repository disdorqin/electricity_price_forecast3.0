# EFM3 前端集成指南 (Frontend Integration Guide)

本指南面向**后续**接入的前端团队。本次交付是 **API-only 后端控制面**，前端**未合并**（保留在 PR #13 `agent/backend-frontend-control-plane` 演示分支）。前端应通过本文件描述的 OpenAPI 契约接入，而非直接调用内部 Python 模块。

---

## 1. 接入契约 (The Contract)

后端通过 FastAPI 自动生成 OpenAPI 3 契约：

- 机器可读：`docs/api/openapi.json`
- 人类可读：`docs/api/API_CONTRACT.md`
- 重新生成（后端环境）：
  ```bash
  backend/.venv/Scripts/python.exe scripts/export_openapi.py
  ```

前端推荐用 [openapi-typescript](https://github.com/openapi-qraft/openapi-typescript) 从 `docs/api/openapi.json` 生成类型安全的客户端：

```bash
npx openapi-typescript docs/api/openapi.json -o src/api/efm3.gen.ts
```
